## Функционал бота
- Фиксация задач и времени.
- Интеграция с Google Календарь/Trello.
- Шифрование данных.
- Генерация отчетов.

## Технологии
- Бэкенд: Python (aiogram).
- БД: PostgreSQL.
- Шифрование: AES.
